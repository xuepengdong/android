    一、客户端源码

    android3目录是《Android Studio开发实战：从零基础到App上线(第3版)》一书的客户端App源码，基于Android Studio Dolphin编写，并在API31（Android 12）上编译通过。
    
    客户端工程的导入办法：打开Android Stuio，依次选择菜单File→Open...，在弹出的文件对话框中选择android3，再单击对话框下方的OK按钮，即可导入客户端的android3工程。
   
    注意：android3\local.properties里面的sdk.dir和ndk.dir要设置为读者电脑上的SDK安装路径和NDK安装路径。


    二、服务端源码

    HttpServer为《Android Studio开发实战：从零基础到App上线(第3版)》一书的服务端源码，基于IDEA和Java8编写。

    服务端工程的导入办法：打开IDEA，依次选择菜单File→Open...，在弹出的文件对话框中选择HttpServer，再单击对话框下方的OK按钮，即可导入服务端的HttpServer工程。
    服务端工程的启动步骤：详见源码包的《服务端工程的使用说明（IDEA版）》。
